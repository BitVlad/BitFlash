# BitFlash

## О программе

BitFlash — программа для записи ISO-образов на USB-флешки.

### Возможности

- Определение подключённых флешек (буква, метка, размер)
- Выбор ISO-образа (в разработке)
- Форматирование в FAT32 (в разработке)
- Запись ISO на флешку (в разработке)

### Технологии

- C#
- Windows Forms
- .NET

### Запуск

1. Открой файл BitFlash.sln в Visual Studio
2. Нажми F5
